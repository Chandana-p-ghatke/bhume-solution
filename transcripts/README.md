# AI Transcripts

This folder records AI assistance used for the BhuMe boundary take-home.

## Codex Desktop

The solution was developed with Codex Desktop in a coding session. The session covered:

- reading the task contract and starter-kit README
- downloading the two village bundles
- running the starter baseline and local scoring
- designing a boundary-aware translation method
- implementing `predict.py`
- scoring both generated `predictions.geojson` files
- preparing and pushing the GitHub repository

See `codex-session.md` for the prompt/response log reconstructed from the working session.
